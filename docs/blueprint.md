# **App Name**: MRT Challenge

## Core Features:

- Firestore Connection: Connect to the 'stations' collection in Firestore to fetch MRT station data.
- Game Mode Selection: Allow users to choose between 'English' and 'Pinyin' game modes.
- Input Validation: Check user input against the 'mrt_station_english' or 'mrt_station_pinyin' field based on the selected game mode.
- Score Tracking: Increase the user's score when a correct station name is entered.
- Station Discovery: Mark stations as 'found' once they have been correctly answered.
- Progress Indication: Visually indicate the player's progress by displaying the number of station answered / total stations.

## Style Guidelines:

- Use SMRT's signature red (#DA291C) for key elements.
- Background color: Off-white or light grey (#F0F0F0) for a clean and modern look.
- Accent color: Dark grey (#434A54) for text and secondary elements.
- Headline font: 'Open Sans' (or similar) for headers, 'Roboto' (or similar) for body text.
- Use flat, simple icons in a similar style to SMRT's branding. Focus on clarity.
- Design a clean and structured layout inspired by SMRT's website. Use clear sections and whitespace.
- Subtle transitions and animations for feedback and interactivity, keeping it professional and smooth.
